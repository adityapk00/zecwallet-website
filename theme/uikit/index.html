Elements

Here is the list of all supported components and UI kit to develop custom blocks using them.

Also, visit our help center.

Layout

List of components which can help you design the layout of your page.

Section

A primary wrapper for your contents with different styles and variations.
Navbar

Responsive and powerful navigation header with branding, navigation, and more.
Topbar

A small component to be placed on top of a page and displays some information.
Sidebar

A container besides your content to display widgets or more information.
Offcanvas

A container to stay outside of the page and come into the page with a button click.

Content

List of components which can help you to develop your content.

Typography

Examples of typography, including headings, body text, lists, and more.
Colors

Paint your website the way you Like it, solid, pale or gradient.
Card

A flexible and extensible content container with several styles and options.
Table

Stylized tables to allow audience grabs the information in a glance.
Image

Opt your images into responsive behavior and add lightweight styles to them.
Video

Examples on how to place a video in your page, from different sources.
Icon

More than 1,000 beautifully crafted icons with some helper classes.
Content

A bunch of most used contents in web development for fast page design.
Utility classes

Some of the most required classes to have for the ease of development.

UI Kit

List of components which can help you to create complex yet engaging content.

Accordion

Allows you to toggle content on your pages with a few classes.
Alert

Provide contextual feedback messages for typical user actions.
Badge

Small and adaptive badge for adding context to just about any content.
Button

Use our custom button styles for actions in forms, dialogs, and more.
Constellation

A beautiful way of making an interaction behind your content.
Count down

A sequence of backward counting to indicate the time remaining for an event.
Count up

A plugin that will count up to a target number at a specified speed and settings.
Dropdown

Toggleable and contextual overlays for displaying lists of links and contents.
Form

Several form control types, layout options, and components to create your forms.
Granim

Create fluid gradients animations by adding a data attribute to a canvas.
Lightbox

A script that displays images and videos by filling the screen.
Google map

Display different kind of Google maps in customizable styles and options.
Miscellaneous

More components such as pagination, tooltip, popover, rating, etc.
Modal

Flexible dialog prompts with the minimum required functionality.
Nav

Several types of navigation markups, styles and variations.
Overlay and scrim

An alpha colored layer to increase contrast of your content.
Popup

Display a small peace of content inside a dialog, without grabbing much attention.
Pricing

Several sample of pricing tables, designed using utility classes.
Process

A set of components to allow you create timelines and display step-by-step processes.
Progress

Stylize the HTML5 progress element with a few extra classes.
Scroll

Few components and script to help you with scrolling to a point or display a content.
Shuffle

Categorize, sort, search and filter a responsive grid of items.
Slider

A slideshow component for cycling through elements like a carousel.
Social

Exhibit your brand on different social networks with an easy to use component.
Tab

Keeps related content in a single container that is shown and hidden through navigation.
Typing text

Enter in any comma-separated string, and watch it type at the speed you've set.
